# AGENTS.md

This repository contains solutions to Exercism exercises in Free Pascal.

## Structure
Each exercise is in its own directory with:
- `Makefile` - Build configuration
- `*.pas` - Pascal source files
- `test.pas` - Test runner
- `TestCases.pas` - Test cases

## Running tests
```bash
cd <exercise>
make test
```

## Submitting solutions
```bash
cd <exercise>
exercism submit
```
