# Exercism Free Pascal Solutions

Solutions to Exercism exercises written in Free Pascal.

## Exercises

- [Darts](darts/) - Calculate points scored in a single toss of a Darts game
- [Hello World](hello-world/) - Classic Hello World exercise
- [Logs](logs/) - Log parsing exercise

## Running Tests

Each exercise has its own `Makefile`. To run tests:

```bash
cd <exercise-directory>
make test
```

## Prerequisites

- Free Pascal Compiler (fpc)
- Exercism CLI
